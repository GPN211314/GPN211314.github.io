一些总结

Posted on 2019-10-08

近来也写了一些 haskell代码，本文就随便说说感受和一些总结

说一下最近学到点什么

学haskell也有一年了，之前从来没接触过函数式的语言，也就最近算写的代码比较多的了，之前一直都是沉浸在haskell的各种特性中，因为从没有接触过所以特别新奇，
刚开始学的时候monad可是把我难倒了，不多bb，写点总结吧。

透镜组

这个看过几次，但是没用过，也没有遇到需要用的

State Monad

没真正用过，看了一遍又一遍，不知道怎么下手，什么时候用，看那些函数作为参数的函数实在费脑筋

Writer Monad

用过一两次，不熟练

(->) r Monad

主要平时会用一下 p <$> f <*> g将两个应用同样参数的函数连起来
这和on有点像，on :: (b -> b -> c) -> (a -> b) -> a -> a -> c，通常和 compre一起用

Monad转换器

基本没用过，主要没写过大项目

暂时先写这么多，等有更深的理解了再来补

写了一段时间后感觉循环都不会写了

感觉也就那么样，没有刚学时那么新奇，但依然有学不完的东西，有些不适合的场景强行用函数式思维确实费劲，但适合的场景用起来真的爽。
最近用队列时看了Data.Sequence，比较好奇是怎么实现的O(1)时间进出队列的，看网上说是复杂的数据结构还有编译器优化。
