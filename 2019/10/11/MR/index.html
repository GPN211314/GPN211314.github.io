Miller-Rabin素性测试算法

Posted on 2019-10-11 | Edited on 2019-10-12
参考博客：https://blog.csdn.net/ECNU_LZJ/article/details/72675595

最近写CodeWar的时候遇到了两个题都是和素数相关的，我才想起上算法课学过的MR素性检测，现在再来回顾一下。
有空再来写写John M. Pollard质因数分解算法和快速幂算法。

前提知识

  • 费马小定理

    若$p$是素数且$a,p$互素，则$a^{p-1} = 1 \ mod \ p$

  • 二次探测定理

    若$p$是素数，且$0 < x < p$则$x^2 \equiv 1 \ mod \ p$的解为$x= \pm 1$

Miller-Rabin测试

  • 给定$n,a$互素
  • 令$n-1=2^ut$，其中$t$为奇数
  • 对$0 \leq i \leq u$，令$x_i=a^{2^it} \ mod \ n$，$(x_{i+1} \equiv x_i^2 \ mod \ n)$
  • 若$x_u \neq 1$，或有$0 \leq i < u$使得$x_i \neq \pm 1$但$x_{i+1} = 1$，则称$a$是$n$为合数的见证（或称$n,a$没通过$M-R$测试）
  • 若$n$是合数，则$n$在$2:n-1$中至少有${n-1} \over 2$个见证

Miller-Rabin算法

  • $Witness(a,n)$

    • 令$n-1 = 2^ut,u \geq1,t$为奇数
    • $x=a^t \ mod \ n$
    • 对$i=1:u$
    • $y=x^2 \ mod \ n$
    • 若$y=1$，且$x\neq1,x\neq n-1$，则返回true
    • 令$x=y$
    • 若$x \neq 1$，则返回true，否则返回false

  • $Miller-Rabin(n,k)$

    • 对$j=1:k$，以均匀分布取$1:n-1$中数作为$a$
    • 若$Witness(a,n)$为true，则返回合数
    • 若所有的$Witness(a,n)$值全为false，此时我们断定$n$为素数

      结语

      MillerRabin算法是一个随机化算法，并不能保证结果完全正确，但其出错概率小于$2^k$，一般当取$k=50$时出错概率已经小到可以忽略不计了。
